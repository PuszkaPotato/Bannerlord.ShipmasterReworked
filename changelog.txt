Version: 1.3.3
Game Versions: v1.4.x
* Fixed an issue, where settings are not applied until mod options are saved. Now the default settings are applied immediately (Reported by LessTalking).
* Added a toggle to apply experience to a Navigator, by default only travelling on seas grants experience to a Navigator.
---------------------------------------------------------------------------------------------------
Version: 1.3.4
Game Versions: v1.3.4 - v1.4.7
* Fixed a crash on older game versions (since the War Sails launch) caused by TaleWorlds changing a naval method's parameters between versions and adding a Navigator property that older builds do not have. The mod now adapts to both older and newer versions.
* Fixed travelling on water granting Shipmaster experience to every hero in the party on older game versions. It is now correctly limited to the party leader (and, on newer versions, the Navigator).
* Harmony patches are now applied individually, so if a future game update changes something the mod hooks into, only that single feature is skipped instead of the whole mod failing to load.
* Lowered the minimum supported game version to v1.3.4 (War Sails launch).
